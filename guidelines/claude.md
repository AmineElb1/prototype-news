# Project Rules

- Never edit anything inside src/imports/.
- The imported Figma code is the design reference only.
- Build new features in src/app.
- Preserve the exact visual appearance unless requested otherwise.
- Prefer reusable React components.
- Use TypeScript strictly.
- Use Tailwind utilities before writing custom CSS.
- Keep App.tsx lightweight by extracting logic into hooks and components.
- Explain the implementation plan before making structural changes.
